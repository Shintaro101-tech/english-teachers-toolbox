# 英語教師のツールボックス / English Teacher's Toolbox

英語教師のためのウェブアプリ集。授業を盛り上げるゲーミフィケーションツールから、教材作成・授業運営の効率化まで。

**運営**: [一般社団法人 Pacific English House](about.html)

## 含まれるアプリ

| アプリ | 状態 | 説明 |
|---|---|---|
| **Quiz Arena** | ✅ 公開中 | Kahootを超えるゲーミフィケーションクイズ。コンボ・パワーアップ・4種の問題形式 |
| **Interactive Slides** | 🚧 開発予定 | Mentimeter風の双方向授業ツール |
| **One-minute Speech** | 🚧 開発予定 | 1分間スピーチのトピックをスロットマシンで決定 |
| **Class Timer** | 🚧 開発予定 | アクティビティ別の授業タイマー |
| **QR Code Generator** | 🚧 開発予定 | 教材URLを瞬時にQRコード化 |
| **Pair Maker** | 🚧 開発予定 | クラスのペアをランダムに決定 |

## 技術スタック

- 純粋な HTML / CSS / JavaScript（バニラ）
- ビルドツール不要、静的ホスティングで動作
- データはブラウザの localStorage に保存（バックエンド不要）

## ローカルでの起動

ファイルを開くだけで動作します。

```bash
open index.html
```

または `index.html` をブラウザにドラッグ&ドロップ。
